# 🍅 Pomodoro Timer

A simple and responsive **Pomodoro Timer** built using **HTML, CSS, and JavaScript**.  
This application helps users stay productive by following the Pomodoro Technique: work for a focused period and then take short breaks.

---

## 📌 About the Project

The Pomodoro Technique is a time-management method that breaks work into intervals (usually **25 minutes**) separated by short breaks.

This project implements that technique in a clean and minimal web application.

Users can:
- Start a focus session
- Pause the timer
- Reset the session
- Take short breaks between work sessions

---

## 🚀 Features

- ⏱ 25-minute focus timer
- ☕ Short break timer
- ▶️ Start / Pause / Reset controls
- 🔔 Alert when the timer ends
- 📱 Responsive design
- 🎨 Simple and clean interface

---

## 🛠 Technologies Used

- **HTML5** – Application structure  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Timer logic and interactivity
